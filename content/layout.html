# Java Coding Conventions for Bellerbys students

# Layout

# Basic

  1. Write only one statement per line.

  2. Write only one declaration per line.

  3. Indentation should be 4 spaces instead of \t.

  4. Use parentheses to make clauses in an expression apparent, as shown in the following code:

    if (statement)
{
    // do something
}
else
{
    // do something
}
